Risk Factors & Stress Tests

Downside Catalysts (Stress Scenarios)

  • iPhone Demand Cliff: If iPhone revenue contracts >5% YoY (vs. base −0%), base case return drops to −4%; probability ~15% if new product cycle disappoints.
  • Tariff Shock: 25% China tariff (geopolitical scenario) compresses gross margin −150 bps; 5-mo return −12% even with flat revenue.
  • Rate Spike: 10Y yield > 5.0% (recession fears) triggers multiple compression (P/E 26x → 24x); downside −6% independent of earnings growth.
  • Services Deceleration: If Services CAGR slips to 8% (antitrust headwinds, saturation), base case return −3%.

Upside Catalysts

  • AI Integration Success: If Apple Intelligence drives iPhone upgrade cycle +3% and Services margin +100 bps, upside +18%.
  • Rate Normalization: If 10Y yield drops to 3.5% (soft landing narrative), P/E expansion to 32x + earnings growth = +15% return.
  • Services Inflection: Services margin +150 bps (pricing power, ecosystem lock-in) alone drives +10% without revenue growth.
